선택자 알아보기

구분 표현 설명
전체 선택자 * 모든 태그에 적용
태그 선택자 태그명 해당 태그에 적용
아이디 선택자 #아이디명 해당 id가 지정된 태그에 적용- 중복 x
클래스 선택자 .클래스명 해당 class가 지정된 태그에 적용-중복 ㅇ
자식 선택자 선택자 > 자식선택자 선택자 바로 아래 위치한 자신선택자에만 적용가능
자손 선택자 선택자 자손선택자 선택자 아래 위치한 모든 자손선택자에 적용(자식도 포함됨)
속성 선택자 선택자[속성=값] 속성이
텍스트 크기 확인

전체선택자를 통해 html내의 모든 파일에 적용시킬 수 있다.

스타일 적용의 우선순위
* < 태그명 < 클래스 < 아이디 < !important
외부스타일 < 내부스타일 < 인라인스타일
